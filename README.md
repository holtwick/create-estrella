# create-estrella

Create a starter project for [Estrella](https://github.com/rsms/estrella), the [esbuild](https://github.com/evanw/esbuild) based build system.

```bash
$ npm init estrella <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

If using Yarn:

```bash
$ yarn create estrella <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
